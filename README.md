# Deloitte Australia – Cyber Security Job Simulation

## Overview

This repository documents my completion of the Deloitte Australia Cyber Security Job Simulation on Forage.

The simulation focused on analyzing web activity logs to identify suspicious user behavior during a suspected security incident.

---

## Scenario

Daikibo Industrials suspected that confidential information had been exposed after unusual activity involving their internal telemetry dashboard.

My task was to review the provided web activity logs and determine whether the incident could have originated directly from the Internet and identify any suspicious user activity.

---

## Objectives

- Analyze web activity logs
- Identify abnormal request patterns
- Detect automated API requests
- Determine whether external Internet access was possible
- Identify the suspicious user involved

---

## Findings

- The dashboard was hosted on an internal network.
- No evidence suggested direct access from the public Internet.
- A user displayed automated API requests at regular intervals.
- The identified suspicious user ID was:

`mdB7yD2dp1BFZPontHBQ1Z`
