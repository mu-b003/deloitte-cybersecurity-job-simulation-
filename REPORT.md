# Investigation Report

## Summary

This investigation was completed as part of the Deloitte Australia Cyber Security Job Simulation available on Forage.

The objective was to inspect a web activity log and determine whether suspicious activity had occurred.

---

## Investigation

The log file was manually reviewed.

Particular attention was given to:

- Login requests
- Dashboard access
- Requests for static resources
- API requests

The expected browsing sequence was compared with the observed activity.

---

## Findings

The investigation found:

- Access originated from internal network addresses.
- No evidence indicated direct access from the public Internet.
- One user generated API requests at consistent time intervals, indicating automated behavior rather than normal dashboard usage.

Suspicious User ID:

`mdB7yD2dp1BFZPontHBQ1Z`

---

## Conclusion

Based on the available evidence, the suspicious activity was most likely generated by an automated process operating from within the internal environment rather than a direct Internet attack.
